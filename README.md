# isbry.com

Personal brand site. Not a portfolio, not a CV — just a honest look at who I am.

Built around a 3D Rubik's cube because I love them. Each face is a different dimension of me: DevOps engineer, AI explorer, learner, builder, human being.

I also love Legos. A lot.

## Stack

Vite + React + Tailwind CSS + Framer Motion

## Run locally

```bash
npm install
npm run dev
```

## Deploy

Automatic on every push to `main` via GitHub Pages.
